# 🦛 HippoWars — Мультиплеерная арена

Пошаговая PvP-игра с бегемотами, рейтинговыми боями, сквад босс-файтами и двумя темами оформления.

## Стек
- **Backend**: Node.js + Express + Socket.io
- **Database**: SQLite (better-sqlite3)
- **Auth**: JWT + bcrypt
- **Frontend**: Vanilla JS + CSS3

---

## 🚀 Запуск локально

```bash
# 1. Перейди в папку backend
cd backend

# 2. Установи зависимости
npm install

# 3. Запусти сервер
npm start
# или в режиме разработки:
npm run dev

# 4. Открой в браузере
# http://localhost:3000
```

## 🌐 Деплой на Railway

1. Зайди на [railway.app](https://railway.app)
2. Создай новый проект → "Deploy from GitHub repo"
3. Выбери репозиторий
4. Railway автоматически определит `npm start` в `/backend`
5. Установи переменные окружения:
   - `JWT_SECRET=your_secret_here`
   - `PORT=3000`
6. Готово! Railway даст публичный URL.

## 🌐 Деплой на Render

1. Зайди на [render.com](https://render.com)
2. New → Web Service → Connect repo
3. Root Directory: `backend`
4. Start command: `npm start`
5. Добавь переменную `JWT_SECRET`

## ⚙️ Переменные окружения

| Переменная | По умолчанию | Описание |
|------------|-------------|----------|
| `PORT` | `3000` | Порт сервера |
| `JWT_SECRET` | `hippowars_secret_2025` | Секрет для JWT (смени в продакшене!) |

## 🎨 Темы оформления

Переключатель находится в **Профиль → Тема**:
- **🌑 Default** — тёмная тема
- **🌍 Африканская** — земляные тона, tribal паттерн
- **🍥 Наруто** — синий/оранжевый, шаринган паттерн

## 📡 WebSocket события

| Событие | Направление | Описание |
|---------|-------------|----------|
| `join_queue` | → сервер | Встать в очередь PvP |
| `leave_queue` | → сервер | Выйти из очереди |
| `match_found` | ← сервер | Матч найден |
| `battle_action` | → сервер | Атака/хил/скилл в PvP |
| `battle_update` | ← сервер | Обновление боя |
| `join_boss_lobby` | → сервер | Войти в лобби босса |
| `boss_lobby_ready` | → сервер | Нажать "Готов" |
| `lobby_updated` | ← сервер | Обновление лобби |
| `send_friend_request` | → сервер | Запрос дружбы |
| `online_count` | ← сервер | Кол-во игроков онлайн |
